É possível estilizar sua página com CSS de três formas:

  1. CSS inline (atributo style)
  2. CSS externo
  3. Tag style

Este texto foi modificado por meio de inline CSS.

Mas podemos encurtar o trabalho por meio da tag style:

Código em Java:

        
        
        public static void main (String[] args) {
            // Imprimir mensagem no terminal
            System.out.println("Hello World!");
            return 1;
        }
